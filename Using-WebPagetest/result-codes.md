# 结果代码
结果代码为0（成功）或99999（内容错误）是成功的测试，所有其他结果都是错误。

| 结果代码 | 描述  |
| :------------ |:---------------|
| 0         | 成功的测试                           |
| 4xx-5xx   | HTTP结果（基页错误）                  |
| 99996     | 测试失败等待指定的DOM元素/结束条件     |
| 99997     | 测试超时（无内容错误）        |
| 99998     | 测试超时（内容错误）        |
| 99999     | 测试成功完成但个别请求失败（内容错误）        |

其他错误代码是Wininet / IE特定的错误代码（通常是大的负数）。识别它们的最好方法是将其转换为十六进制。 如果您在找到它们时向我们发送新的错误代码，我们将它们添加到表中。
